# Cloud Speaker

## Installation

Download Cloud Speaker code from the official repository. Follow these steps to get started:

1. Clone the repository to your local machine
2. Install all required dependencies
3. Configure your environment variables

## Compile and build

To compile and build the Cloud Speaker software, run `project_build.bat` and choose options in this order: 1 → 2 → 3

```
Main menu
====================

1. build clean                 ②
2. build app                   ③
3. packet firmware (zip file)  ④
4. packet APP OTA file(bin file)
5. packet DIFF SYS OTA file(bin file)
6. packet DIFF APP SYS OTA file(bin file)
7. set device type             ①
8. quit

note: if you select item 6 and item 7, kernel will be rebuilded , don't do it if u

pc>dir: D:\project\cloud60-sdk-env-jio\project1

please select item number, then click enter button.
```
